INTRODUCTION
------------

Current Maintainer: Jeremy Lichtman <jeremy.l@myplanetdigital.com>

CyberSource SASOP (Secure Acceptance Silent Order POST) integration for the 
Drupal Commerce payment and checkout system. Currently supports credit card 
payments on the checkout form via SASOP using the latest Commerce version 
as of July 4, 2013.

INSTALLATION
------------

 * Install the CyberSource SASOP module.
 * Then enable the Rule for CyberSource SASOP via Store > Configuration > 
   Payment settings and edit the enabling action to use your CyberSource API 
   credentials.
